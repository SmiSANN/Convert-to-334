# Converter334
ver 1.0
## 何ができるの？
世の中の自然数を334で表すことができる

### 例
- 1 = 3*3-4 - (3/3)*4
- 2 = 3+3-4
- 3 = 3*3-4 - 3+3-4
- 4 = (3/3)*4
- 5 = 3*3-4
...
- 334 = 334
- etc...

## ルール
- 最低1組の[3,3,4]を使用する
- [3,3,4]の並びを崩さない
- 1組では表せない場合、組数を増やして良い
- 組と組の間も四則演算のみ
- 計算方法は四則演算のみ(modやべき乗は使用しない)
- ()を使用した計算順序の指定は[3,3,4]の組を崩さなければ使用可能
- 複数の組を跨いだ()の使用は不可能
- 指定の値を表す最小の計算式にする

## 流れ(準備)
- 334の取りえる全ての数値を配列へ
- 上記の配列で取りえる全ての計算結果を配列へ

## 流れ
- 入力値nを受け取る
- 準備で使用した計算結果を使用
- 
- 1~nまでを334の組で新しく作るか、元あるものを再利用して作成




## 使用方法
関数 `convert_to_334()`の引き数に自然数を与えると、計算式を文字列が返却される
